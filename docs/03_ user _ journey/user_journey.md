    Action: ouverture de l'application web 
    Problème potentiel: manque de précision

    Action: lecture des posts et  accès a l'onglet de réduction

    Action: écriture et envoie d'un post
    Problème potentiel: interface ilisible 
    Solution UX: interface visible 

## Etape 4 : interaction
    Action: like et commentaire
    Problème potentiel: trop d'option, ilisible
    Solution UX: icone simple et compréhensible
